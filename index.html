Conexión con Google Drive

El dashboard lee los 3 Excel mensuales (COMPRAS, RECETAS POR ART, RECETAS POR INSUMOS) desde una carpeta pública de Drive. Los datos se procesan en el navegador, sin servidor.

Restringí la key a Drive API y al dominio de GitHub Pages.

URL: drive.google.com/drive/folders/ESTE_ES_EL_ID. Compartir como "Cualquiera con el enlace".

Archivos detectados

Cómo configurar

  1. Andá a Google Cloud Console → APIs & Credentials y creá una API Key.
  2. Habilitá la Google Drive API en tu proyecto.
  3. (Recomendado) Restringí la key por HTTP referrer al dominio de GitHub Pages.
  4. Subí los 3 Excel del mes a una carpeta de Drive y compartila como "Cualquiera con el enlace puede ver".
  5. Copiá el ID de la carpeta (la parte final de la URL).
  6. Pegá API Key + Folder ID acá y dale Guardar.
